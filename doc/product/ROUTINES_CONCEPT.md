# Routines -- Product Concept

> Audience: product + design + engineering
>
> Scope: product purpose and UX intent for the Routines screen.

## Primary purpose

"Routines help users support their values with repeatable actions, while
tracking stats to learn what is working."

## Description line (UI copy)

"Support your values with routines, and use stats to improve what works."

## Key question it answers

"Which routines are helping my values, and what should I adjust?"

## What it shows

- Flexible and scheduled routines with a clear cadence (daily, weekly, monthly).
- Progress and trajectory context so users can spot momentum changes.
- A leading value icon that makes the value link obvious.
